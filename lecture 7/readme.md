TALAL KHAN — Portfolio (Lecture 06)

Professional personal portfolio developed for the Web Fundamentals course at Savonia University of Applied Sciences. This project demonstrates semantic HTML structure, responsive CSS design, accessibility improvements, JavaScript interactivity, and external API integration.

Features

Feature	Implementation
Skills Section	8 skill cards displayed using a responsive CSS grid layout
Projects Section	3 project cards with category tags, project descriptions, and GitHub links
Theme Toggle	Light/Dark mode using isDark state + body.classList.toggle('dark')
Persistent Theme	User theme preference saved using localStorage (portfolio_theme)
Click Counter	Interactive JavaScript button with live counter update
Last Updated Date	Automatically generated using JavaScript with new Date()
External Data Demo	fetch() + async/await + try/catch + response.ok for safe API handling
Accessibility	Semantic HTML, skip link, ARIA labels, accessible navigation and media
