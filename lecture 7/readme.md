TALAL KHAN — Portfolio (Lecture 06)

Personal portfolio built for the Web Fundamentals course at Savonia University of Applied Sciences.

Features
Feature	Implementation
Skills section	8 skill cards in a responsive CSS grid
Projects section	3 project cards with tag, description and GitHub link
Theme toggle	isDark state + body.classList.toggle('dark')
Persistent theme	localStorage key portfolio_theme
Last updated date	Generated by JS via new Date() on every load
External Data Demo	fetch() + async/await + try/catch + response.ok

API used
GET https://jsonplaceholder.typicode.com/users/1

Displays: Name, Email, Company, Username, Phone, Website, City.

Live site
https://mdparvez-alam.github.io/Lecture-06/

Run locally
Open index.html in any browser — no build step needed.
